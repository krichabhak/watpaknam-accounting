# 📖 คู่มือการใช้งาน (User Guide)

## 🎯 วัตถุประสงค์ของระบบ

ระบบนี้ถูกออกแบบเพื่อ:
1. ✅ บันทึกและจัดการรายรับ-รายจ่ายของวัด
2. ✅ สร้างรายงานทางการเงินแบบเรียลไทม์
3. ✅ ติดตามเงินสะสมและยอดคงเหลือ
4. ✅ ให้ความโปร่งใสในการเงิน

## 🔐 Step 1: การเข้าใช้ระบบ

### สมัครสมาชิกครั้งแรก
```
1. เปิด https://watpaknam-accounting.vercel.app
2. คลิก "สมัครสมาชิก" 
3. กรอก:
   - ชื่อ-นามสกุล
   - อีเมลที่ใช้งานจริง
   - รหัสผ่าน (อย่างน้อย 8 ตัวอักษร)
   - ยืนยันรหัสผ่าน
4. คลิก "สมัครสมาชิก"
5. ตรวจสอบอีเมล verification (หากจำเป็น)
6. กลับไปที่ Login
```

### เข้าสู่ระบบ
```
1. ใส่อีเมลและรหัสผ่าน
2. คลิก "เข้าสู่ระบบ"
3. ระบบจะพาไปยัง Dashboard
```

## 📊 Step 2: Dashboard - ภาพรวม

หลังเข้าระบบจะเห็น:

### 3 สถิติหลัก (Top Cards)
```
┌─────────────────────────────────────┐
│  รายรับทั้งหมด  │  รายจ่ายทั้งหมด  │  ยอดคงเหลือ  │
│   ฿ 50,000      │   ฿ 20,000       │   ฿ 30,000    │
└─────────────────────────────────────┘
```

- **สีเขียว**: รายรับ (Income)
- **สีแดง**: รายจ่าย (Expense)  
- **สีน้ำเงิน**: ยอดคงเหลือ (Balance)

### 2 กราฟแสดงข้อมูล

#### เลียบ Bar Chart (แท่ง)
- แสดงเปรียบเทียบรายรับ vs รายจ่าย
- แท่งเขียว = รายรับ
- แท่งแดง = รายจ่าย

#### ใด้มว Pie Chart (วงกลม)
- แสดงสัดส่วนรายรับต่อรายจ่าย
- สัดส่วนเปอร์เซ็นต์

### ตารางรายการล่าสุด
- แสดง 5 รายการล่าสุด
- วันที่ | รายการ | ประเภท | จำนวนเงิน

## 💰 Step 3: บันทึกรายการ

### วิธีบันทึกรายรับ/รายจ่าย

```
1. คลิกปุ่ม "บันทึกรายการ" (สีส้ม) ที่ Sidebar ซ้าย
2. Modal (หน้าต่างป็อปอัป) จะปรากฏ
3. กรอกข้อมูล:

   ┌─────────────────────────────────┐
   │ ประเภท    [▼ รายรับ]           │  เลือก "รายรับ" หรือ "รายจ่าย"
   │ ชื่อรายการ [_______________]    │  เช่น "บริจาค", "ค่าไฟ"
   │ จำนวนเงิน  [_______________]    │  เช่น "5000"
   └─────────────────────────────────┘

4. คลิก "บันทึก" เพื่อบันทึก
5. คลิก "ยกเลิก" เพื่อปิดหากเปลี่ยนใจ
```

### ตัวอย่างการบันทึก

**ตัวอย่าง 1: บริจาคจากสมาชิก**
```
ประเภท:    รายรับ (+)
ชื่อรายการ: บริจาคจากสมาชิก
จำนวนเงิน:  5,000
```

**ตัวอย่าง 2: ค่าไฟฟ้า**
```
ประเภท:    รายจ่าย (-)
ชื่อรายการ: ค่าไฟฟ้า
จำนวนเงิน:  2,500
```

**ตัวอย่าง 3: ค่าซ่อมแซมอาคาร**
```
ประเภท:    รายจ่าย (-)
ชื่อรายการ: ซ่อมแซมหลังคา
จำนวนเงิน:  15,000
```

## 📋 Step 4: ดูรายการทั้งหมด

### Sidebar Navigation
```
┌─────────────────────┐
│  ① ภาพรวม (Default) │  ← ดูตัวเลขสรุปและกราฟ
│  ② รายการ          │  ← ดูรายการทั้งหมด
│  ③ รายงาน          │  ← ดูการวิเคราะห์
│                     │
│ [บันทึกรายการ]     │
└─────────────────────┘
```

### Tab "รายการ"
```
1. คลิก "รายการ" บน Sidebar
2. จะแสดงตารางรายการทั้งหมด:

   ┌──────┬──────────────────┬──────────┬────────────┐
   │วันที่ │ รายการ          │ ประเภท   │ จำนวนเงิน │
   ├──────┼──────────────────┼──────────┼────────────┤
   │ 13/7 │ บริจาคสมาชิก    │ รายรับ   │ +฿5,000   │
   │ 12/7 │ ค่าไฟฟ้า        │ รายจ่าย  │ -฿2,500   │
   │ 11/7 │ ซ่อมแซมหลังคา  │ รายจ่าย  │ -฿15,000  │
   └──────┴──────────────────┴──────────┴────────────┘
```

## 📈 Step 5: ดูรายงานและการวิเคราะห์

### Tab "รายงาน"
```
1. คลิก "รายงาน" บน Sidebar
2. จะแสดง 2 ส่วน:

   ┌─────────────────────┐  ┌──────────────────┐
   │ สรุปรายรับ-รายจ่าย │  │ สถิติ            │
   │ ─────────────────  │  │ ────────────────  │
   │ รายรับ: ฿ 50,000   │  │ จำนวนรายการ: 15  │
   │ รายจ่าย: ฿ 20,000  │  │ เปอร์เซ็นต์รายรับ │
   │ ยอดสุทธิ: ฿ 30,000 │  │            71.4%  │
   └─────────────────────┘  └──────────────────┘
```

## 🔐 Step 6: ออกจากระบบ

```

2. คลิก "ออกจากระบบ"
3. ระบบจะพาไปยัง Login Page
```

## ⚡ Tips & Tricks

### 💡 Tip 1: ใช้ชื่อรายการที่ชัดเจน
```
❌ ไม่ดี: "เงิน", "ค่าใช้จ่าย"
✅ ดี:    "บริจาคจากสมาชิก", "ค่าเก้าอี้พลาสติก"
```

### 💡 Tip 2: ตรวจสอบยอดคงเหลือเป็นประจำ
- ตรวจสอบ Dashboard ทุกสัปดาห์
- บันทึกรายการอยู่เสมอ

### 💡 Tip 3: เตรียมข้อมูลก่อนบันทึก
- เขียนลงสมุดไว้ก่อน
- เก็บใบสำคัญไว้
- บันทึกวันที่ให้ถูกต้อง
